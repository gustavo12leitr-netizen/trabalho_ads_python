# Relatório de Configuração e Exercícios - ADS

Este projeto contém uma série de exercícios de lógica de programação e monitoramento de sistemas utilizando Python, desenvolvidos para a faculdade de ADS.

## 🛠️ Configuração do Ambiente

Para o desenvolvimento, segui os seguintes passos de configuração:

1. **Instalação do VS Code:** Utilizado como IDE principal por sua versatilidade e integração com Git.
2. **Interpretador Python:** Utilizei a versão **3.14.3** instalada via `uv`. No VS Code, selecionei o interpretador através do comando `Ctrl + Shift + P` > `Python: Select Interpreter`.
3. **Extensões:** Instalei a extensão oficial do Python (Microsoft) para suporte a IntelliSense e Depuração.

## 📂 Estrutura do Projeto

Cada exercício foi organizado em um arquivo separado para facilitar a leitura e manutenção:
- `ex01_saudacao.py`
- `ex02_soma.py`
- (E assim por diante...)

## 🚀 Versionamento com GitHub

O código é versionado utilizando o **Git**. O fluxo de trabalho consiste em:
1. `git init`: Inicializa o repositório local.
2. `git add .`: Adiciona as alterações para o próximo commit.
3. `git commit -m "Mensagem"`: Salva o estado atual do código.
4. `git push`: Sincroniza o código local com o repositório remoto no GitHub.
